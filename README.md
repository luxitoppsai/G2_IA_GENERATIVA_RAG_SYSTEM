## Requisitos

- Python 3.10 o superior
- Jupyter Notebook o VS Code con soporte para notebooks
- Dependencias que se usan dentro del notebook

## Uso rápido

1. Abre el archivo `G2_Sistema_RAG_IA_GENERATIVA_MDS.ipynb`.
2. Instala las librerías que solicite el notebook.
3. Ejecuta las celdas en orden.

## Objetivo

Construir y evaluar un prototipo de sistema RAG que combine recuperación de información y generación de respuestas con modelos de lenguaje.
